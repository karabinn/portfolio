Booki

Création d’une page d’accueil responsive pour une agence de voyage
Lien vers le projet sur GitHub

Objectif :

Le projet Booki consistait à intégrer la page d’accueil d’un site pour une agence de voyage en partant de maquettes Figma. L’objectif était de créer une version responsive, fidèle aux maquettes, tout en respectant les normes d’accessibilité. Le tout a été développé uniquement avec HTML5 et CSS3, sans framework ni JavaScript.

Technologies utilisées :

HTML5, CSS3, Git, GitHub, Figma

Défis :

Le principal défi était d’assurer une intégration responsive mobile-first, sans outils automatisés. Il fallait aussi respecter les maquettes Figma tout en veillant à la conformité aux standards d’accessibilité.

Solutions apportées :

  • Utilisation de l'approche mobile-first pour garantir l’adaptabilité du site sur tous les écrans.
  • Emploi de Flexbox et de media queries pour garantir une interface fluide et fidèle aux maquettes.
  • Structuration sémantique du HTML pour améliorer l’accessibilité et le SEO.
  • Organisation des styles CSS pour assurer une bonne maintenabilité du code.
  • Gestion de version avec Git, en veillant à avoir des commits propres et en utilisant des branches.

Ce que j'ai appris :

CCe projet m’a permis de maîtriser l’intégration responsive en partant de maquettes multi-supports, tout en appliquant les bonnes pratiques HTML/CSS. J’ai aussi appris à gérer un projet avec Git et GitHub, tout en prenant en compte les règles d’accessibilité dès la conception.

  • Intégration HTML & CSS
  • Approche mobile-first
  • Fidélité aux maquettes Figma
  • Versionning Git & GitHub
