Mon Vieux Grimoire

Développement du back-end d’un site de notation de livres
Lien vers le projet sur GitHub

Objectif

Le but de ce projet était de développer le back-end d'un site de notation de livres. Ce système devait permettre aux utilisateurs de s'inscrire, de se connecter, d'ajouter des livres, de noter ces livres et d’interagir avec les autres utilisateurs via une API RESTful sécurisée.

Technologies utilisées

Ce projet a été réalisé avec Node.js pour la gestion du serveur, Express.js pour l'API RESTful, MongoDB pour la base de données NoSQL, et Mongoose pour la modélisation des données. Pour la sécurité, nous avons utilisé les JSON Web Tokens (JWT) pour l’authentification et des bonnes pratiques de Green Code pour optimiser l’impact environnemental du projet.

Défis rencontrés

Les principaux défis étaient de garantir la sécurité des données des utilisateurs, d’assurer des performances optimales malgré une possible forte charge de données et d'optimiser la consommation énergétique du projet tout en maintenant une bonne qualité de service.

Solutions apportées

  • Implémentation d'un système d'authentification sécurisé via JSON Web Tokens (JWT), garantissant une gestion fiable des sessions utilisateurs.
  • Utilisation de Mongoose pour la gestion des données, permettant une modélisation claire et évolutive des documents dans MongoDB.
  • Optimisation des performances du projet en suivant des bonnes pratiques de Green Code, réduisant l'impact écologique du développement tout en garantissant des temps de réponse rapides.
  • Mise en place d'une architecture MVC (Modèle-Vue-Contrôleur) pour assurer la séparation des préoccupations et faciliter la maintenance du code.
  • Configuration d'un système de gestion des erreurs afin d'améliorer la robustesse et la sécurité du back-end en cas de mauvaises requêtes ou d'attaques potentielles.

Ce que j'ai appris

Ce projet m’a permis de renforcer mes compétences en développement back-end, notamment en gestion d'API RESTful avec Node.js et Express. J’ai également appris à gérer la sécurité des applications web via JWT, à optimiser les performances d’un serveur et à réduire l’empreinte écologique du code grâce aux pratiques de Green Code. Cela m’a permis de mieux comprendre l'architecture moderne des serveurs et de développer des compétences importantes pour la gestion d’une application scalable et sécurisée.

  • Développement d’API REST avec Node.js et Express
  • Modélisation des données avec Mongoose et MongoDB
  • Mise en œuvre de l’authentification et sécurité des données
  • Respect des principes Green Code
