Sophie Bluel

Création d’une page web dynamique pour une architecte d’intérieur
Lien vers le projet sur GitHub

Objectif :

Le projet Sophie Bluel consistait à créer une page web dynamique pour une architecte d’intérieur. L’objectif était de concevoir une interface interactive permettant de gérer les réalisations de l'architecte et de créer un espace d'administration sécurisé pour l’ajout de nouveaux projets. La page a été développée en utilisant JavaScript pour manipuler le DOM et gérer les événements utilisateurs, tout en intégrant des appels à une API pour la gestion dynamique des données.

Technologies utilisées :

HTML5, CSS3, JavaScript, API, Git, GitHub

Défis :

Le principal défi était de créer une interface dynamique qui permettait non seulement de manipuler des données en temps réel, mais aussi de gérer un espace d'administration sécurisé. Il fallait aussi assurer la communication fluide avec une API pour récupérer et envoyer des informations tout en garantissant une expérience utilisateur fluide et intuitive.

Solutions apportées

  • Manipulation du DOM et gestion des événements utilisateurs pour une interface interactive.
  • Utilisation de JavaScript natif pour la gestion des formulaires et l’interaction avec l’API.
  • Création d’un espace d’administration sécurisé avec une page de connexion et une modale d’ajout de médias.
  • Intégration des appels API pour récupérer et envoyer des données de manière dynamique.
  • Versionnement du code avec Git, hébergement sur GitHub pour un suivi optimal du projet.

Ce que j’ai appris

Ce projet m’a permis de consolider mes compétences en JavaScript et d'apprendre à gérer des interactions dynamiques sur une page web. J’ai également approfondi ma compréhension des appels API et de leur utilisation en temps réel. Ce projet m’a aussi permis de mettre en place un espace sécurisé d’administration, un aspect essentiel pour tout site interactif.

  • DOM et événements JavaScript
  • Communication avec une API
  • Interface d’administration dynamique
  • Gestion de projet avec GitHub
