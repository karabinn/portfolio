KASA

Réalisation du front-end d’une application de location immobilière en React
Lien vers le projet sur GitHub

Objectif :

Le projet consistait à développer le front-end d’une application de location immobilière en utilisant React. L’objectif était de créer une interface moderne et responsive, avec une navigation fluide grâce à React Router. L’application devait être modulable et facilement maintenable, en utilisant des composants réutilisables.

Défis :

Le principal défi résidait dans la gestion de l’interactivité au sein d’une SPA (Single Page Application), tout en maintenant une interface fluide et réactive. Il fallait également garantir une navigation entre plusieurs pages sans rechargement, ainsi que l’intégration de styles dynamiques avec SASS.

Solutions apportées :

  • Développement d'une SPA (Single Page Application) avec React.
  • Utilisation de React Router pour gérer la navigation entre les différentes pages de l’application.
  • Création de composants réutilisables pour assurer la modularité et l’évolutivité de l’application.
  • Utilisation de SASS pour styliser l'interface de manière efficace et maintenable.
  • Intégration d’animations et de transitions CSS pour une expérience utilisateur fluide.

Ce que j’ai appris :

Ce projet m’a permis de renforcer mes compétences en React, notamment en gestion de l’état de l’application et du routage avec React Router. J’ai également approfondi l’utilisation de SASS pour une gestion efficace des styles et intégré des animations CSS pour rendre l’interface plus interactive et engageante.

  • Développement d'une SPA avec React
  • Navigation multi-pages avec React Router
  • Intégration CSS avec SASS
  • Animations et transitions CSS
