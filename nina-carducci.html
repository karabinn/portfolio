Débug et optimisation SEO d'un site de photographe
Lien vers le projet sur GitHub

Objectif

Améliorer les performances, l’accessibilité et le référencement naturel (SEO) d’un site web existant pour une photographe professionnelle. Le travail s’est basé sur un audit technique complet, avec mise en œuvre concrète des recommandations, dans le cadre de ma formation de développeur web.

Technologies utilisées :

HTML, CSS, JavaScript, Chrome DevTools, Lighthouse, Wave

Défis :

L’une des principales difficultés a été d’identifier précisément les éléments responsables du mauvais score SEO et des problèmes d’accessibilité. Certains problèmes n’étaient pas immédiatement visibles à l’œil nu, mais impactaient fortement l’expérience utilisateur et le référencement.

Solutions apportées :

  • Réalisation d’un audit technique avec Lighthouse, Wave, et les DevTools de Chrome pour analyser les performances, le code HTML/CSS et l’accessibilité.
  • Optimisation des images pour améliorer les temps de chargement.
  • Refonte partielle du HTML afin de respecter une structure sémantique cohérente (balises header, main, article, etc.).
  • Réduction du code inutile et amélioration du CSS/JS pour un rendu plus fluide.
  • Mise en œuvre de bonnes pratiques SEO (balises meta, attributs alt, titres hiérarchisés, etc.).
  • Rédaction d’un rapport comparatif avant/après pour mesurer l’impact des optimisations.

Ce que j'ai appris :

Ce projet m’a permis d’approfondir mes connaissances en débogage web, performance front-end , et accessibilité numérique. J’ai appris à mener un audit technique, à prioriser les corrections et à mesurer concrètement leur impact. Cela a également renforcé ma capacité à structurer un projet d’optimisation dans une démarche qualité.

  • Optimisation des performances d’un site web
  • Débogage à l’aide des Chrome DevTools
  • Amélioration de l’accessibilité et du SEO
  • Rédaction d’un cahier de recette pour tester un site
